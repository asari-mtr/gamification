# 第Ⅰ部　基礎編 — 設計判断の土台をつくる

ゲーミフィケーションを「ポイントを貼ること」で終わらせないために、まず**動機づけの本質**を理解する。第Ⅰ部は、以降のすべての設計判断が立脚する土台を固める。

## この部で扱うこと

- **第1章　ゲーミフィケーションとは何か** — 定義・隣接概念との違い・歴史・典型的な誤解、そして「そもそもこの課題に向くか」の見極め。
- **第2章　なぜ人は動くのか — 動機づけの心理学** — 自己決定理論（SDT）・フロー・行動経済学・習慣形成・アンダーマイニング効果。本書の理論的中核。
- **第3章　ゲームの構成要素を分解する** — PBLの限界、MDA、3層モデル、進行システム、フィードバックループ。
- **第4章　プレイヤーを知る — ユーザータイプ論** — Bartle・HEXAD・多動機設計。

## この部を読むと

「人はなぜ動くのか」を言語化でき、あらゆる施策を**有能感・自律性・関係性**の観点で点検できるようになる。これが第Ⅱ部以降の設計の共通言語になる。
