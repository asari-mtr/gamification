# 第10章　ヘルスケア・行動変容

!!! abstract "この章のねらい"
    健康行動の変容は、ゲーミフィケーションが社会的に最も価値を生みうる領域であると同時に、失敗が**健康被害**に直結しうる領域でもある。本章では、フィットネス・メンタルヘルス・疾患管理の実践を、行動変容理論との接続、成功例（Apple Watch・Strava・Zombies, Run!）、そしてランダム化比較試験（RCT）が示すエビデンスとともに扱う。

## 10.1 なぜ健康行動は続かないのか

健康行動（運動・食事・服薬）が続かない理由は、行動経済学（第2章）でよく説明できる。

- **現在バイアス** — 運動の苦痛は「今」、健康の便益は「将来」。割に合わないと感じる。
- **フィードバックの遅延** — 努力の成果（体重・健康指標の改善）が見えるまで時間がかかる。
- **抽象的な目標** — 「健康になる」は漠然としていて行動に変換しにくい。

ゲーミフィケーションは、これらに対して「**即時の報酬・可視化された進捗・具体的な日々の目標**」を提供することで、将来の便益を「今やる理由」に翻訳する。健康分野でゲーミフィケーションが有望視される理論的根拠はここにある。

## 10.2 行動変容理論との接続

ヘルスケアでは、ゲーミフィケーション単独ではなく、確立された**行動変容理論**と組み合わせるのが定石。

- **トランスセオレティカルモデル（変化の段階）** — 無関心期→関心期→準備期→実行期→維持期。段階に応じて適切な仕掛けが異なる（無関心期に競争を持ち込んでも響かない）。
- **目標設定理論** — 具体的で挑戦的だが達成可能な目標が行動を促す。SMART目標とゲーミフィケーションの目標設計は親和的。
- **社会的認知理論** — 自己効力感（≒有能感、第2章）が行動継続の鍵。小さな成功体験の積み重ねが効く。
- **COM-B モデル** — 行動には Capability（能力）・Opportunity（機会）・Motivation（動機）が必要。Fogg の B=MAP（第2章）と同系統。動機だけ上げても能力・機会がなければ行動は起きない。

!!! tip "理論との併用が成否を分ける"
    健康分野の研究では、**行動変容理論に裏打ちされた設計**のほうが、単にポイントを付けただけの設計より効果が高い傾向が報告されている。「どの段階の・誰の・どの行動を・なぜ変えるのか」を理論で詰めることが、医療現場での信頼にもつながる。

## 10.3 フィットネス・運動

最も普及した領域。ウェアラブルとの統合で「行動の自動計測→即時フィードバック」が実現した。

### Apple Watch のアクティビティリング

- **3つのリング**（ムーブ/エクササイズ/スタンド）を「閉じる」というシンプルで強力な目標可視化。
- **進捗バーの円環版**で、目標勾配効果（第2章）が常時働く。
- **ストリーク・実績バッジ**で継続を促進。
- **シンプルさ**が秀逸——3つの明確な目標だけに絞り、認知負荷が低い。

### Strava — ソーシャル × 競争

- ランニング/サイクリングの記録を共有し、**セグメント**（特定区間）でランキングを競う。
- 競争（KOM/QOM＝区間王者）と社会的承認（kudos）を組み合わせる。
- **コミュニティ・関係性**（第2章）が継続の大きな要因。「見られている」ことが動機になる。

### Zombies, Run! — 物語による没入

- ランニングに「ゾンビから逃げ、物資を集めて拠点を再建する」物語を付与（第6章）。
- 単調な運動に**意味の文脈**を与え、内発的動機に近い没入を生む好例。

!!! warning "フィットネス・ゲーミフィケーションの注意"
    - **過剰な競争**は、無理な運動・怪我・燃え尽きを招きうる。
    - **ストリークの強迫**が、体調不良時にも無理をさせる危険。救済設計（第6章）が健康文脈では特に重要。
    - 計測できる行動（歩数）に偏ると、計測しにくいが重要な行動（筋トレ・休養）が軽視される（測定の歪み、第8章）。

## 10.4 メンタルヘルス・習慣化

- **メンタルヘルスアプリ** — 瞑想（Headspace 等）、気分記録、認知行動療法（CBT）ベースのアプリで、継続をストリーク・進捗・リマインダーで支援。
- **習慣化アプリ** — 習慣のチェックイン、連続記録、進捗の可視化。Fogg の「Tiny Habits」（小さく始める）と相性が良い。
- **留意点** — メンタルヘルス領域では、ストリークの**喪失が自己否定**につながりうる。「途切れた＝失敗・ダメな自分」という感覚を生まないフレーミングが極めて重要。損失回避の使い方を誤ると、助けるべき人を傷つける。

!!! danger "脆弱なユーザーへの配慮"
    メンタルヘルス・依存症・慢性疾患などの領域では、ユーザーが**心理的に脆弱**な状態にあることが多い。競争・損失回避・社会的比較といった仕掛けは、健常者には動機づけでも、脆弱なユーザーには**有害**になりうる。ブラックハット（第5章）の使用は原則避け、有能感・自律性・関係性をやさしく支える設計に徹する。倫理レビュー（第14章）が不可欠。

## 10.5 疾患管理・服薬アドヒアランス

- **慢性疾患管理** — 糖尿病の血糖記録、高血圧の測定習慣などを、記録の継続・目標達成で支援。
- **服薬アドヒアランス** — 服薬の記録・リマインダー・連続達成で、飲み忘れを減らす。
- **小児医療** — 子ども向けに、治療や検査をゲーム化して不安を軽減し協力を引き出す（例:採血・リハビリのゲーム化）。
- **留意点** — 医療行為に関わるため、**安全性・正確性・医療者の関与**が前提。エンゲージメントのために医学的妥当性を犠牲にしてはならない。

## 10.6 エビデンス — RCTが示すもの

ヘルスケアは医療分野ゆえ、**ランダム化比較試験（RCT）**による比較的質の高い検証が行われている。

- **身体活動（歩数等）** — ゲーミフィケーション＋社会的支援（家族・チーム）の介入が、歩数を有意に増やしたRCTが複数ある。とくに**ソーシャル要素（関係性）や損失回避フレーミングを組み合わせた設計**で効果が報告されている。
- **効果の持続** — 介入期間中は効果が出ても、**介入終了後に元に戻る**（リバウンド）ことが多い。内発的動機への橋渡しができているかが、持続の分かれ目。
- **質のばらつき** — 研究の質・期間・対象がまちまちで、メタ分析でも結論には幅がある（第15章）。「効くという証拠はあるが、万能ではない」が公正な要約。

!!! example "損失回避を使った歩数介入"
    行動経済学の知見を使い、「目標歩数を達成できなかったら、あらかじめ預けたお金（または付与済みのポイント）を失う」という**損失フレーミング**の介入が、「達成したら報酬を得る」利得フレーミングより効果的だったとする研究がある。同じ報酬額でも、**失う恐れ**のほうが行動を強く駆動する（損失回避、第2章）。ただし、これを健康の文脈でどこまで使うかは倫理的判断を伴う。

## この章のまとめ

- 健康行動が続かない理由（現在バイアス・遅延フィードバック・抽象的目標）に、ゲーミフィケーションは「即時報酬・進捗可視化・日々の目標」で応える。
- 単独でなく、**行動変容理論（変化の段階・目標設定・社会的認知・COM-B）と併用**するのが定石。
- フィットネスでは Apple Watch（シンプルな目標可視化）、Strava（社会×競争）、Zombies, Run!（物語）が好例。
- メンタル・疾患領域では**脆弱なユーザー**への配慮が必須。ブラックハットは避け、やさしく支える。
- RCTは「効くが、社会的要素・損失回避との併用が鍵」「介入後にリバウンドしがち」と示す。**持続には内発的動機への橋渡し**が要る。

次章では、組織と収益が絡む**ビジネス・組織**分野へ。動機づけと労働倫理の緊張関係が前景化する。
