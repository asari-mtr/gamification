# 第Ⅲ部　応用編 — 分野別の実装パターンと事例

設計原則が、各分野で**どう具体化され、何が成功し何が失敗したか**を、事例とともに見る。分野固有の論点（教育の学習成果、健康の脆弱性、労働の倫理、公共の公平性）を押さえる。

## この部で扱うこと

- **第9章　教育・学習** — Duolingo / Kahoot! の解剖、学習成果のエビデンス。
- **第10章　ヘルスケア・行動変容** — 行動変容理論との接続、RCTの知見、脆弱なユーザーへの配慮。
- **第11章　ビジネス・組織** — ロイヤルティ、HR、営業、クラウドソーシング、労働倫理。
- **第12章　社会・公共・サステナビリティ** — 省エネ、金融、市民参加、ナッジとの境界。

## この部を読むと

自分の分野に近い事例から、**移植可能な設計パターンと避けるべき落とし穴**を学べる。
