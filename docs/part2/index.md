# 第Ⅱ部　設計編 — 実装できる設計に落とす

第Ⅰ部の理論を、**手順とカタログ**に変える。場当たり的に要素を足すのではなく、目的から逆算して、実装できる設計に落とし込む。

## この部で扱うこと

- **第5章　設計フレームワーク** — Werbach 6D、Chou Octalysis、ワークショップ進行テンプレート。
- **第6章　メカニクスのカタログ** — 達成・競争・協力・進行・経済・物語の6系統と、選定ディシジョンツリー。
- **第7章　体験の設計** — プレイヤージャーニー、オンボーディング、難易度曲線、リテンション、報酬スケジュール、エンドゲーム。
- **第8章　設計のアンチパターン** — 失敗の典型と回避のチェックリスト。

## この部を読むと

目標から要素までを一貫して設計でき、**典型的な失敗を事前に回避**できるようになる。
