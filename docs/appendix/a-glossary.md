# 付録A　用語集

本書に登場する主要な用語を五十音・アルファベット順に整理する。各項目末尾に主な解説章を示す。

## あ行

**アンダーマイニング効果（undermining / overjustification effect）**
: もともと内発的に楽しんでいた活動に外的報酬を与えると、かえって内発的動機が低下する現象。安易なポイント付与の危険の根拠。（第2章）

**アンチパターン**
: 繰り返し失敗を生む典型的な悪い設計。ポインティフィケーション、競争の逆効果、測定の歪みなど。（第8章）

**アダプティブ（適応的）ゲーミフィケーション**
: ユーザーのタイプ・行動・習熟・文脈に応じて、提示するメカニクスや難易度を出し分ける設計。（第16章）

**MDA フレームワーク**
: ゲームを Mechanics（メカニクス）・Dynamics（ダイナミクス）・Aesthetics（エステティクス）の3層で捉える枠組み。「設計はM→A、体験はA→M」の非対称が要点。（第3章）

**Octalysis（オクタリシス）**
: Yu-kai Chou による8つのコアドライブで動機を捉えるフレームワーク。左脳/右脳、ホワイトハット/ブラックハットの軸を持つ。（第5章）

**オンボーディング**
: ユーザーが使い方と楽しさを学ぶ初期フェーズ。早い成功・段階的開示・初期進捗が鍵。（第7章）

## か行

**外発的動機づけ**
: 活動の外にある結果（報酬・評価・罰の回避）のための動機。質に差があり、内発へ統合されうる。（第2章）

**ガードレール指標**
: 主指標を追う際に、悪化を見張るための監視指標（質・満足度・解約率など）。測定の歪みの早期検出に使う。（第13章）

**グッドハートの法則**
: 「指標が目標になると、良い指標でなくなる」。測定の歪みの理論的背景。（第8・13章）

**現在バイアス**
: 将来の大きな便益より、目先の小さな報酬を過大評価する傾向。健康・学習・貯蓄が続かない一因。（第2章）

**コアドライブ**
: Octalysis における8つの動機の源泉。（第5章）

**コンポーネント**
: 3層モデルの最下層。ポイント・バッジ・レベル等の具体的な実装パーツ。PBLはここに属する。（第3章）

## さ行

**自己決定理論（SDT: Self-Determination Theory）**
: 有能感・自律性・関係性の3つの基本的心理欲求を中核とする動機づけ理論。本書の設計の土台。（第2章）

**シリアスゲーム**
: 娯楽以外の目的（教育・訓練等）のために作られた、完全なゲーム。ゲーミフィケーションとは区別される。（第1章）

**社会的証明（social proof）**
: 「他の人もやっている」が行動を促す心理。省エネ行動の促進などで利用。（第2・12章）

**ストリーク（連続記録）**
: 連続して行動した日数などの記録。損失回避を使った強力な継続装置。救済（フリーズ）設計が重要。（第6章）

**スキャフォルディング（足場かけ）**
: 初期は補助を厚くし、習熟に応じて外していく支援設計。教育学由来。（第7章）

## た行

**ダークパターン**
: ユーザーを欺いたり、本人の利益に反する行動へ誘導する設計。ゲーミフィケーションの仕掛けが転化しうる。（第14章）

**ダイナミクス**
: 3層モデルの最上層。制約・感情・物語・進行・関係性といった体験の大きな枠組み。（第3章）

**adaptive difficulty（適応的難易度）**
: ユーザーの実力に応じて難易度を自動調整し、フロー状態を維持する仕組み。（第7・16章）

## な行

**内発的動機づけ**
: 活動それ自体が面白い・満足できるから行う動機。持続性が高い。（第2章）

**ナッジ（nudge）**
: 選択を強制せず、より良い行動へそっと誘導する設計思想。ゲーミフィケーションと重なる。（第2・12章）

**ノベルティ効果**
: 新しい仕掛けが、目新しさだけで一時的に指標を上げ、やがて減衰する現象。測定の最大の落とし穴。（第7・13章）

## は行

**PBL（Points, Badges, Leaderboards）**
: ポイント・バッジ・リーダーボード。最も多用されるが最も浅い層の要素。過信は「ポインティフィケーション」を招く。（第3章）

**HEXAD**
: Marczewski による6つのユーザータイプ（フィランソロピスト・ソーシャライザー・フリースピリット・アチーバー・プレイヤー・ディスラプター）。SDTと接続。（第4章）

**Fogg Behavior Model（B=MAP）**
: 行動は動機（Motivation）・能力（Ability）・きっかけ（Prompt）が揃ったとき起きるとするモデル。（第2章）

**フロー**
: 挑戦とスキルが釣り合うとき生じる、活動への完全な没入状態。Csikszentmihalyi。（第2章）

**ブラックハット / ホワイトハット**
: Octalysis の軸。ブラックハット（希少性・予測不能性・損失回避）は焦り・不安で動かす。ホワイトハット（意義・達成・創造）は前向きに力づける。（第5章）

**プレイヤージャーニー**
: 発見→オンボーディング→習熟→熟達という、時間軸でのユーザー体験。（第7章）

**Play-to-Earn（P2E）**
: 「遊んで稼ぐ」Web3ゲームモデル。外発的報酬の弊害・経済の持続不可能性を露呈した。（第17章）

**ポインティフィケーション（pointsification）**
: 動機づけ設計を欠いたまま PBL だけを貼り付ける安直な手法への批判語。（第1・8章）

## ま行

**Manipulation Matrix**
: Nir Eyal による設計者の倫理自己点検。「自分も使うか」「ユーザーの生活を改善するか」の2軸。（第14章）

**meaningful gamification（意味のゲーミフィケーション）**
: 内発的動機・自己実現・現実とのつながりに根ざした、人間中心の設計方向。（第18章）

**目標勾配効果**
: ゴールに近づくほど努力が増す効果。進捗バーや初期進捗付与が効く理由。（第2章）

## や・ら・わ行

**ルートボックス / ガチャ**
: 有料の変動報酬。射幸性が高く、各国でギャンブル規制の対象。（第6・14章）

**リテンション**
: ユーザーの定着・継続。習慣化を時間軸で設計すること。（第7章）

**リーダーボード**
: 順位表。競争動機を生むが、下位ユーザーの離脱を招く両義的な仕掛け。（第3・8章）

**6D**
: Werbach による6ステップ設計プロセス（目標→行動→プレイヤー→ループ→楽しさ→要素）。（第5章）

**変動比率強化**
: 不定回数ごとに報酬を与える強化スケジュール。最も強力で消去されにくい＝中毒的。ガチャの基礎。（第2・7章）

**報酬スケジュール**
: 報酬を与えるタイミングのパターン（連続・固定間隔・変動比率など）。Skinner由来。（第7章）
