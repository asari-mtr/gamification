# 付録

本編の内容を、実務で参照しやすい資料としてまとめる。

## 収録

- **付録A　用語集** — 本書の主要用語を解説章とともに整理。
- **付録B　デザイン・テンプレート集** — 6Dワークシート、SDT点検シート、Octalysis点検シート、体験フロー設計シート、効果測定プラン、倫理レビューなど、コピーして使えるテンプレート。

- **付録D　主要文献・研究リスト** — さらに深く学ぶための文献。

## 使い方

実際のプロジェクトでは、**付録B のテンプレート**を出発点に設計を進め、判断に迷ったら本編の該当章へ戻る、という往復が効率的。
