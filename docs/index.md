# ゲーミフィケーション実践大全

> 基礎理論から実践設計、応用分野、効果測定・倫理、そして最新研究まで——
> **「使えるゲーミフィケーション」を体系的に学ぶ**ための実務者・デザイナー向けテキスト。

ゲーミフィケーションは「ポイントとバッジを付けること」ではありません。人間の動機づけのメカニズムを理解し、行動が自然と続く体験を設計する営みです。本書は、心理学的な裏付けを土台にしながら、実際のプロダクトに落とし込める設計判断ができるようになることを目標にしています。

## 想定する読者

- プロダクト/サービスにゲーミフィケーションを取り入れたい **企画者・デザイナー・エンジニア**
- 学習・健康・組織などで **行動変容** を設計したい実務者
- ゲーミフィケーションを **体系的に理解** したい人

## 進め方のおすすめ

はじめての方は第Ⅰ部から順に読むことを推奨します。すでに基礎がある方は、第Ⅱ部（設計）や第Ⅲ部（応用）から必要な箇所をつまみ読みしても構いません。各章末には実務で使えるチェックリストやテンプレートを配置しています。
