# 第Ⅳ部　計測と倫理 — 続けられる仕組みにする

ゲーミフィケーションを「作って終わり」にしないために、**効果を測り、倫理を守る**。本書を通じて繰り返し現れた2つの主題を、正面から掘り下げる。

## この部で扱うこと

- **第13章　効果をどう測るか** — KPI設計、A/Bテスト、行動データ分析、ノベルティ効果の見抜き方、ダッシュボード設計。
- **第14章　倫理・ダークサイド** — ダークパターン、依存・射幸性、労働の監視、操作と支援を分かつ境界、倫理レビュー。

## この部を読むと

「効いているか」を科学的に判断でき、「誰のためか」を倫理的に点検できる——成熟した実務家に不可欠な2つの規律が身につく。
